Images

Image component can be responsive to their parent Element. (They never become wider than their parent.)

Example of Responsive Image

You have just add responsive-images to make image responsive. This class applies max-width: 100%; and height: 100%; to the image so that it scales with the parent width.

Example of Round Image

You need to add class img-round to the image in order to make the image round shaped.
