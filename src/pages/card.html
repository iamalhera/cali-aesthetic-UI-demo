    CARD

    Cards are pre-built containers that provides a short and crisp information to the user through text, images, links, and buttons.

    Horizontal Card

    Create a horizontal card by using the code below. You can add class card , card-horizontal and card-image-horizomntal for the img tag.

    Code Example :

    Oziya Medicine

    Price : 200 Rs

    Its a medice to give to the patients of diabetes

    Vertical Card

    Vertical card are pre build containers to be used in any website with basic editable elements. You can add class card , card-vertical-thin , card-vertical-thick and card-image-vertical for the img tag.

    Code Example :

    Taazgi

    Price : 300 Rs.

    A medicine to improve memory and to advances neuroplasticity
